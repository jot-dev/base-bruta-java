# Base Bruta - Java Exercises

This repository serves as a personal collection of foundational Java exercises. The goal here is to consolidate my understanding of core logic, control structures, and algorithm problem-solving.

## 📂 Project Structure

The project is organized by topic to maintain a clear learning path:

- **`Condicoes/`**: Exercises focused on decision-making and flow control.
  - Includes: `IfElse`, `SwitchCase`, `DoWhile`, and `Scannerdousuario`.
- **`Tiposdedados/`**: Exploration of Java primitive and non-primitive data types.
  - Includes: `DadosPrimitivos` and `DadosNãoPrimitivos`.
- **Root Files**:
  - `Calculadoraimc.java`: A practical BMI calculator application.
  - `ProjetosDeEstudo.Desafionaruto.java`: An applied logic challenge.

## 🛠 Tech Stack

- **Language**: Java
- **IDE**: IntelliJ IDEA
- **Version Control**: Git & GitHub
